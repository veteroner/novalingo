## TestFlight (iOS) — Dağıtım Adımları

1. `pnpm build && npx cap sync ios`
2. Xcode'da `ios/App/App.xcworkspace` aç → **Product → Archive**
3. Organizer → **Distribute App → App Store Connect → Upload**
4. App Store Connect → **TestFlight** → build processed olduğunda **Internal Testing** grubuna ekle
5. External Testing için **Test Information** + Beta App Review doldur, ardından public link üret
6. 4-9 yaş çocukların ebeveynlerinden oluşan pilot grubu davet et (maksimum 10,000 external tester)

## Google Play Internal Testing — Dağıtım Adımları

1. `pnpm build && npx cap sync android`
2. `cd android && ./gradlew bundleRelease` → `app/build/outputs/bundle/release/app-release.aab`
3. Google Play Console → **Testing → Internal testing** → **Create new release** → AAB yükle
4. **Tester list**: Google Group veya e-posta listesi → opt-in URL paylaş
5. Release notes (TR + EN) ekle, review **staged rollout %10** ile başlat
6. Feedback sonrası **Closed testing** → **Open testing** → **Production** akışı
